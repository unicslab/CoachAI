## Overview

CoachAI is a real-time intelligent tennis coaching system built on edge AI. Designed for players of all levels, CoachAI delivers professional-grade performance analysis directly on-device — no internet connection required on the court, no lag, just instant and actionable coaching feedback.

CoachAI is developed and maintained by the **Ubiquitous Networking and Intelligent Computing System (UNICS) Lab** at the New Jersey Institute of Technology (NJIT).

---

## Features

| Feature | Description |
|---|---|
| 🎾 **Real-Time Stroke Analysis** | Instant feedback on forehand, backhand, serve, and volleys with live pose detection |
| 👣 **Footwork & Movement Tracking** | Live analysis of court coverage, positioning, and movement patterns |
| 📚 **Structured Lessons** | Progressive lesson plans from basics to mastery across all stroke types |
| 🏆 **XP & Badges** | Earn experience points and badges as you complete lessons and level up |
| 📊 **Progress Dashboard** | Session-by-session tracking across technique, fitness, and strategy |
| 🔒 **On-Device Privacy** | All AI processing happens on-device — your data never leaves your phone |
